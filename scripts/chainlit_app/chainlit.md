# George

A cognitive AI with persistent memory, adaptive attention, and executive function.

---

Type naturally to chat. Use the **`/`** icon for explicit commands.

| Command | What it does |
|---------|-------------|
| `/memory <query>` | Search across all memory systems |
| `/memory <system>` | Browse `stm` · `ltm` · `episodic` · `semantic` · `prospective` · `procedural` |
| `/reminders` | List active reminders |
| `/remind <minutes> <text>` | Set a reminder |
| `/goals` | List active goals |
| `/goal <title>` | Create a goal |
| `/dream` | Run a consolidation cycle |
| `/reflect` | Run a reflection report |
| `/learning` | Learning metrics and experiments |
| `/metacog` | Narrative self-report |
| `/metacog --raw` | Full diagnostic dashboard |
